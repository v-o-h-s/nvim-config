# Introduction

I have been using Neovim (LazyVim, specifically), and I have enjoyed the experience a lot. It is one of the best code editors I have used.

I have been customizing it for a while, and this repository contains my personal Neovim configuration. Feel free to use it as inspiration or adapt it to your own setup.
